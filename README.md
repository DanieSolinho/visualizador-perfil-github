# Visualizador de Perfil do GitHub

Este projeto é uma aplicação web simples para visualizar perfis de usuários do GitHub. Basta digitar o nome de usuário e a aplicação buscará e exibirá informações como nome, avatar, seguidores, seguindo, repositórios e mais.

## Funcionalidades

*   Buscar perfis de usuários do GitHub.
*   Exibir informações básicas do perfil (avatar, nome, bio, seguidores, etc.).
*   Listar os repositórios públicos do usuário.
*   Indicador de carregamento durante a busca.
*   Tratamento básico de erros para usuários não encontrados.

## Tecnologias Utilizadas

*   **HTML5**: Estrutura da página web.
*   **CSS3**: Estilização e responsividade da interface.
*   **JavaScript (ES6+)**: Lógica de busca e manipulação do DOM.
*   **GitHub API**: Para buscar os dados dos perfis e repositórios.
*   **Devicon**: Para o ícone do GitHub.

## Como Executar Localmente

Siga os passos abaixo para configurar e executar o projeto em sua máquina local:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/visualizador-perfil-github.git
    cd visualizador-perfil-github
    ```
    *(Nota: Substitua `https://github.com/seu-usuario/visualizador-perfil-github.git` pelo URL real do repositório, se diferente.)*

2.  **Abra o arquivo `index.html`:**
    Este é um projeto front-end puro. Você pode simplesmente abrir o arquivo `index.html` em seu navegador web preferido.

    Alternativamente, para uma experiência de desenvolvimento, você pode usar uma extensão como "Live Server" no VS Code ou similar para servir o arquivo localmente.

## Como Usar

1.  Abra a aplicação em seu navegador.
2.  No campo de busca, digite o nome de usuário do GitHub que deseja visualizar.
3.  Pressione "Buscar" ou a tecla Enter.
4.  As informações do perfil e os repositórios serão exibidos na tela.
